## 📌 Deskripsi Repository

Repository ini digunakan untuk menyimpan tugas akhir (TA) mata kuliah **Sistem Informasi Web**.  
Project yang dibuat merupakan pengembangan website secara bertahap dengan perbaikan (fixing) dan penambahan fitur interaktif.

---

## 📂 Struktur Tugas

###  [Week 2](./Week2/TA_week2)  
Pada week ini melanjutkan project dari minggu sebelumnya dengan penambahan fitur dan perbaikan.

Fitur yang ditambahkan:
- **Fitur Pengelolaan Stok**  
  Jika tombol beli diklik, maka stok akan berkurang secara real-time dan muncul alert konfirmasi
- **Fitur Tema (Local Storage)**  
  Tersedia tombol **Dark Mode / Light Mode** pada navbar dan tersimpan otomatis
- **Fitur Wishlist / Keranjang**  
  Menggunakan **Session Storage** dan ditampilkan dalam bentuk **modal**
- Melakukan fixing dari tugas sebelumnya

---

###  [Week 3](./Week3/TA_week2)  
Pada week ini dilakukan perbaikan bug dari minggu sebelumnya serta penambahan fitur baru.

Fitur yang ditambahkan:
- **Sistem Login dan Logout**
- Validasi login (menampilkan **alert jika username/password salah**)
- Alert saat **logout berhasil**
- Perbaikan bug dari fitur sebelumnya

---

##  Tentang Project

Project ini merupakan pengembangan website berbasis **HTML, CSS, dan JavaScript** dengan fokus pada:
- Pengelolaan interaksi user
- Penyimpanan data menggunakan **Local Storage** dan **Session Storage**
- Simulasi sistem login sederhana

---

## Catatan

- Project ini merupakan bagian dari tugas akhir (TA)
- Setiap week berisi pengembangan dari minggu sebelumnya
- Masih dalam tahap pembelajaran
